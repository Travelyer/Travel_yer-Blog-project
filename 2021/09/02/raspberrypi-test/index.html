树莓派并行尝试（尚未完成）

发表于2021-09-02|更新于2021-09-02

一、准备

  1. 2 × 树莓派3B+ （包含必须的电源和框架）
  2. 2 × 16GB SD卡
  3. 1 × 8段路由器
  4. 3 × 网线（实际2根足用）
  5. 2 × 显示器、键盘和鼠标
  6. 2 × USB数据线

二、安装系统

  1. 进入https://win32diskimager.org/ 下载Win32DiskImager，下载树莓派系统（选择包含常用应用的版本）
  2. 使用读卡器为树莓派安装系统

三、下载与安装

  1. 在两个树莓派上安装SSH、keychain、Network File System、MPI

  2. sudo apt-get install openssh-server (SSH)

    sudo apt-get install keychain
    sudo apt-get install nfs-kernel-server（在raspberrypi1上安装即可）

    sudo apt-get install nfs-common

  3. cd /home/pi

    mkdir mpich3

    cd mpich3

    mkdir build install

    wget http://www.mpich.org/static/downloads/3.2.1/mpich-3.2.1.tar.gz

    tar xvfz mpich-3.2.1.tar.gz

    cd mpich-3.2.1

    /home/pi/mpich3/mpich-3.2.1/configure -prefix=/home/pi/mpich3/install

    make

    make install (MPI)

四、静态IP

  1. 进入并编辑Ip

    cd /etc/network

    sudo nano interfaces

    # interfaces (5) file used by ifup (8) and ifdown (8)

    # Include files from /etc/network/interfaces.d:

    source-directory /etc/network/interfaces.d

    auto eth0

    iface eth0 inet static

    address 192.168.1.10

    gateway 192.168.1.1

    netmask 255.255.255.0
    （在树莓派1号上 192.168.1.10，2号上 192.168.1.10）

    （静态ip会导致无法联网，先完成下载安装再进行）

    如果更改失败可禁用dhcpcd，并启动网络

    sudo systemctl disable dhcpcd

    sudo systemctl enable networking
